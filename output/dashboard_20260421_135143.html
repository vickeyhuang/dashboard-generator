Dashboard Explanations

📊 Status Mapping Explanation

The status categories in this dashboard are mapped from Jira statuses as follows:

  • To Do = To Do + Backlog + Prioritized + Up Next + Selected for Development
  • In Progress = In Progress + In Review + Waiting for Release + Dev + STG + PRD
  • Blocked = Blocked
  • Done = Done + Completed + Closed + Resolved + Released

This mapping provides a simplified view of project progress by consolidating related statuses into four main categories for clearer tracking and reporting.

🎯 Completion Percentage Circles Explanation

The colored circles on the right side of the Delivery Progress chart indicate the completion percentage for each project:

High Completion Rate (≥80%) - On track or ahead of schedule
Medium Completion Rate (50-79%) - Steady progress
Low Completion Rate (<50%) - Needs attention

Completion percentage is calculated as: Done Issues / Total Issues × 100%
